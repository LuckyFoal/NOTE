# 位运算实战要点

首先要明确一点，这里使用位运算符的可以直接是二进制形式，也可以是十进制形式；因为位运算符也是会在十进制转为二进制后运算

判断奇偶
x % 2 == 1 -----> (x & 1) == 1；如果最后二进制位=1则是奇数
x %2 == 0 ------> (x & 1) == 0
乘 / 除2
x /= 2 -----> x >>= 1；实际上编译器底层也会优化成位运算
mid = (left + right) / 2 ------> mid = (left + right) >> 1
x 与 x
x & -x = 1
x & ~x = 0
判断是否是 2 的幂次方：n & (n - 1) == 0
